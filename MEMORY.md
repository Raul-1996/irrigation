
## Zones UI v2 (31.03.2026)

### Что сделано
- Hunter-style карточки зон на странице Статус (заменили таблицу + старые карточки)
- Табы групп с фильтрацией, скрыта группа 999
- Аккордеон: свёрнуто=#id имя тип длительность след.полив, развёрнуто=детали+actions
- Bottom sheet редактирование (имя, тип, длительность, группа)
- Inline +/- длительности (api.put с debounce)
- Таймер обратного отсчёта при поливе + progress bar
- Поиск зон, stats bar
- Async загрузка next-watering (не блокирует рендер)
- 59 UI тестов (26 template + 33 functional)
- Все тесты GREEN на локальном и на WB-8 ARM

- CSRF: нужно использовать api.put() а не fetch() с ручными headers
- Кнопки touch target: min-height 44px для мобилки
